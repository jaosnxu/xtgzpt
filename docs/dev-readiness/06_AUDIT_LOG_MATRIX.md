# AUDIT LOG MATRIX

## 1. 审计原则

所有关键动作必须留痕。

审计记录必须：

- 不可物理删除
- 不可原地修改
- 可追加纠错记录
- 可按对象查看
- 可按用户查看
- 可按时间查看
- 权限受控

## 12. 审计验收

任意一个关键动作，测试必须能回答：

- 谁做的
- 什么时间做的
- 对哪个对象做的
- 做之前是什么
- 做之后是什么
- 为什么做
- 成功还是失败
- 有没有 AI 参与
- AI 用了哪个框架版本
