## 验收标准

- 上传创建独立文件元数据、版本和对象绑定记录
- 文件绑定已有业务对象，当前支持 project / task / chat_thread / knowledge_item / project_memory
- 预览和下载同时校验后端文件权限和来源对象继承权限
- 无权限用户不能预览或下载，响应不泄露文件名或敏感对象名
- 正式流程文件归档/作废而非物理删除
- 上传、预览、下载、归档、权限拒绝和 AI 文件引用均写审计
- AI 使用文件前必须通过当前用户 reference_ai 权限检查
- 前端文件控件只在既有相关页面/组件内出现
- 文件页面状态覆盖 normal / empty / loading / no-permission / error / archived / version history
