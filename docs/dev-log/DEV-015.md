- `/approvals`、`/approvals/:id`、同意、驳回、退回、转交、加签接口
- 后端校验审批权限和当前节点处理人
- 非当前处理人不能处理；无权限详情访问返回非泄露 404
- 同意最终节点写回合同 `approved`
- 驳回写回合同 `rejected`
- 退回写回合同 `revision_required`
- 工作台 pending approval 和系统内通知接入
- 前端审批工作区只在既有“审批”一级菜单内
- 审批实例、节点、动作历史和合同写回审计
