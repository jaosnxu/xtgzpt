## 验收标准

- AI Framework 配置只允许系统管理员或超级管理员
- AI Run 读取按 `read_ai_runs` 和来源对象权限过滤
- AI Run 包含 framework id、framework version、scenario、actor、organization、source object、source ids、input/output snapshot refs、context source ids、status、failure class、retry policy metadata、created/completed time
- 每条 AI 输出能追踪来源证据和输入/输出快照
- 人类采纳、驳回和修改 AI 输出均有记录和审计
- 无权限读取 AI Run 不泄露输入、输出、来源证据或业务对象内容
- 前端不新增 AI 中心或其他冻结外一级菜单
