## 发现与处理

### DEV-024-PX-001 首页像开发验收页

现象：

- 首页包含权限摘要、审计边界、设置嵌套展示和阶段编号，用户看起来像技术验收页。

处理：

- 改为今日待处理、项目运行、合同审批、组织范围和系统提醒。
- 删除首页权限摘要、审计边界和设置嵌套面板。

### DEV-024-PX-002 系统设置不是生产后台结构

现象：

- 设置中心主要展示权限数量和运行证据，缺少正式后台分组。

处理：

- 增加组织与账号、角色与数据范围、审批与文件、审计与运行四组设置卡片。
- 保留 AI 治理和运行状态，但改为正式配置语境。

### DEV-024-PX-003 开发文案残留

现象：

- Browser 验收发现工作台运行状态仍包含阶段编号，系统设置直接显示内部策略版本 `seed-dev-016`。

处理：

- API 运行状态文案改成业务描述。
- 前端展示策略版本时转换为业务可读格式。
- 增加 `app-copy.test.ts` 防止回归。

### DEV-024-PX-004 Browser 截图通道超时

现象：

- Browser 插件 `Page.captureScreenshot` 连续对全页、视口、小区域截图超时。

处理：

- 没有阻塞业务验收。
- 保存逐页 DOM 验收 JSON 到 `docs/dev-log/DEV-024-browser-evidence.json`。
- 该问题记录为工具通道问题，不影响页面可访问和交互验证。
