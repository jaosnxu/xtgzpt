## 范围

- 知识状态支持 `draft`、`submitted_for_review`、`published`、`rejected`、`archived`
- AI 知识草稿人工确认后只进入 `submitted_for_review`，不能自动发布
- 知识管理员或具备发布权限的人类管理员发布、驳回、归档知识
- 知识版本记录 author、reviewer、version、status、timestamps 和 source evidence
- 发布、驳回、归档、创建版本和知识问答均写审计
- 知识问答每条结果返回 source evidence
- 未发布、被驳回、已归档或无权限来源不进入检索、证据或 AI 输入上下文
- 搜索保持本地可测试全文检索，文档化后续本地 vector adapter 策略，不接外部搜索服务
- 前端知识库页面展示审核队列、版本历史、来源证据和发布/驳回/归档控件
- 新增 PostgreSQL 兼容迁移资产 `0007_knowledge_productionization.sql`
