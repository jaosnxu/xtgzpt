## 验收标准

- `/knowledge/query` 不再返回 `not_implemented`
- 空查询返回 `query_required`
- 查询只返回当前账号可见的正式知识和项目记忆
- 查询结果包含 `knowledge_item` 和 `project_memory`
- 查询结果包含相关分、匹配字段和来源消息
- 生成新 AI 草稿时自动带入可见知识 / 记忆上下文
- AI 草稿记录 `contextSourceIds`
- 会话绑定项目后，摘要确认生成的项目记忆绑定同一项目
- 知识库页面可以直接检索上下文
- 检索动作写审计

## 发现与处理

### DEV-008-P1-001 项目记忆未绑定当前项目导致检索被项目过滤

现象：

- 浏览器验证中，知识检索只返回 `knowledge_item`
- `project_memory` 已入库但没有出现在检索结果

原因：

- 前端查询带入当前 `selectedProjectId`
- 聊天摘要确认入库时没有把当前项目传给后端
- 项目记忆以 `projectId=null` 保存，被项目过滤排除

处理：

- 创建聊天会话时带入当前项目关联
- 摘要草稿确认时带入当前项目
- 重新浏览器验证，检索结果同时返回 `knowledge_item` 和 `project_memory`

### DEV-008-P2-002 阶段显示残留 DEV-007

现象：

- 首页指标卡和聊天模块局部 eyebrow 仍显示 DEV-007

处理：

- 统一更新为 DEV-008
- 浏览器刷新后确认首页、聊天页、知识页阶段显示一致

### DEV-008-P3-003 Browser 截图接口超时

现象：

- `tab.screenshot` 连续超时

处理：

- 改用 macOS `screencapture` 保存浏览器验证截图
- 截图文件已保存为 PNG

### DEV-008-P2-004 知识检索项目范围不够显性

现象：

- 知识检索会按当前选中项目传入 `projectId`
- 页面没有显示当前检索范围，测试时容易误判为检索缺失

处理：

- 知识库检索区增加“当前检索范围”
- 有选中项目时显示项目名称
- 没有选中项目时显示“全部可见范围”
