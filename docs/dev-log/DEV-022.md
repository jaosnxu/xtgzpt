## 审计证据

- `apps/api/src/runtime-store.ts`：`memory` / `file` / `postgres` mode selection 已存在；PostgreSQL mode 校验 URL、schema/table identifier 和 document id；driver-backed adapter 使用 `pg` Pool、`runtime_data_documents` 初始化/读取/写入和 checksum 条件更新；读写失败或 checksum 冲突显式失败。
- `apps/api/src/index.ts`：API startup 等待 `store.ready`，写入路径等待 async `store.save()`，close 时调用 `store.close()`。

- GitHub branch protection 已启用。
- Required checks `lint`、`typecheck`、`test`、`build-smoke`、`audit` 全绿。

- PostgreSQL backup、file-to-postgres backfill 和隔离恢复演练已完成。
- Production smoke 使用专用测试账号完成。
