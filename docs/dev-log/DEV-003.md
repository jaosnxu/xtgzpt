## Reviewer 处理记录

独立 reviewer 发现：

- P2：拒绝记录直接信任 `x-request-id`，可能被写入敏感猜测字符串
- P2：`/modules/:module` 对未知模块会把猜测模块名写入拒绝记录
- P2：`esbuild` override 作用范围过大

处理结果：

- 拒绝记录只使用服务端 request id，不再写入客户端 `x-request-id`
- 未知模块先返回 404，不进入权限拒绝记录
- `esbuild` override 缩小到 Vite 依赖范围
- 补充回归测试：恶意 `x-request-id` 不进入拒绝记录；未知模块名不进入拒绝记录
