## 验收标准

- 普通用户进入系统能看到自己要处理什么
- 管理员和普通用户看到不同工作台内容
- 普通用户看不到系统设置，系统管理员不默认拥有普通用户业务项目
- 待办、审批、合同确认、AI 结果、无权限、系统状态通知均有系统内展示
- AI 结果必须由人工确认后才能入库
- 页面状态覆盖 normal、empty、loading、no-permission、error、AI_Generating、AI_Failed、expired、archived
- 1440 / 1280 / 960 宽度有响应式约束且不新增冻结外一级菜单
